# 🏦 Bank Loan EMI Calculator & Analysis Tool

An advanced Excel-based Bank Loan Management System featuring 
EMI calculation, complete amortization scheduling, multi-loan 
comparison, and an interactive financial dashboard.

---

## 📋 Features

- Instant EMI calculation using PMT formula
- Complete 60-month Amortization Schedule
- Month-by-month Principal & Interest breakdown
- Cumulative Principal and Interest tracking
- Loan Paid % tracker per month
- 3-loan simultaneous comparison tool
- Auto recommendation of best loan option
- Processing fee inclusion in total cost
- Interactive dashboard with 4 charts

---

## 📊 Excel Formulas Used

| Formula | Purpose |
|---------|---------|
| PMT | Monthly EMI calculation |
| IPMT | Interest component per month |
| PPMT | Principal component per month |
| CUMIPMT | Cumulative interest paid |
| CUMPRINC | Cumulative principal paid |
| EDATE | Loan payment date generation |
| IFERROR | Error handling |
| MIN + IF | Best loan recommendation |
